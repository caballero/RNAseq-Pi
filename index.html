RNAseq-Pi - A general, flexible, modular and scalable RNAseq Pipeline.

Copyright (C) 2011 Juan Caballero [Institute for Systems Biology]

Description

RNAseq-Pi is a collection of programs (modules) to run transcriptomic analysis in RNAseq samples.

The modules can be classified as: a) Composition filters b) Homology filters c) Sequence mapping d) Special operations

many steps can be ommited or in different order (modular) or can use diferent aligners like bowtie, bwa or blat (flexibility).
